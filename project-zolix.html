Zolix: Customer Relationship Management

Tech Stack: React / Node.js / Express / MongoDB / Tailwind CSS

Project Overview

Zolix is a highly advanced, production-ready SaaS Customer Relationship Management (CRM) platform designed to simplify operations and automate growth for modern teams. It features a complete end-to-end architecture from secure authentication to a powerful workflow automation engine, all wrapped in a visually stunning dark-mode-first premium user interface.

Key Features

  • Intelligent Lead Management: Advanced Kanban boards and data tables to track, organize, and transition leads through the sales pipeline. Features CSV import/export capabilities.
  • Workflow Automation Engine: A customizable rules engine that allows users to trigger automated actions (like sending emails or creating tasks) when specific conditions are met (e.g., a lead status changes).
  • Role-Based Access Control (RBAC): Enterprise-grade security with strict role boundaries. Different interfaces and permissions exist for 'owners', 'admins', and 'managers'.
  • Premium Dark UI: Built with Tailwind CSS and Lucide Icons, featuring advanced glassmorphism, dynamic gradients, mesh backgrounds, and highly polished micro-animations.
  • Invitation-Based Onboarding: Secure tokenized invite system allowing administrators to invite new team members directly via email links.
  • Full Workspace Isolation: Multi-tenant architecture ensuring data privacy across different organizations using the platform.

Technical Architecture

Zolix is built on a scalable MERN stack, with a focus on clean code, reusable components, and secure data handling.

Layer Technology Used Purpose
Frontend React, React Router, Vite Component-based SPA architecture with fast HMR.
Styling Tailwind CSS & Lucide React Utility-first styling for complex responsive layouts and sleek iconography.
Backend API Node.js & Express RESTful JSON API with modular route controllers.
Database MongoDB Atlas & Mongoose Relational-like schemas in a NoSQL environment (linking Leads, Users, Automations).
Security JWT & Bcrypt Stateless authentication via JSON Web Tokens and HTTP-only cookie strategies.

Component Breakdown

1. Frontend Experience

  • Dynamic Routing: Utilizes React Router DOM to manage public landing pages, authentication flows, and protected dashboard routes.
  • Advanced Layouts: Implements a persistent dashboard shell (Sidebar + Header) that seamlessly wraps nested child routes.
  • Glassmorphic Aesthetics: Extensive use of backdrop-blur utilities combined with semi-transparent backgrounds to create a highly premium, modern software feel.

2. Backend Infrastructure

  • Modular Design: The backend is strictly organized into domains (users, leads, auth, automations) keeping routes, controllers, and services cleanly separated.
  • Automation Engine: A custom-built background service that evaluates lead status changes against user-defined rules, executing tasks or sending emails automatically.

Deployment Workflow

The application is built for cloud production:

  • Frontend: Continuously deployed via Netlify, utilizing _redirects to support React Router's SPA routing.
  • Backend: Hosted on Render, managing live API requests with environment variables tightly securing database URIs and JWT secrets.
