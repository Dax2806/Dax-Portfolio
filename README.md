## 📖 Overview

This repository contains the source code for my personal developer portfolio. Built to be lightweight, performant, and visually striking, the portfolio utilizes a dark-mode brutalist aesthetic combined with smooth micro-interactions. It serves as a central hub for my resume, experience, and detailed case studies of my shipped projects.

## ✨ Key Features

- **Brutalist Aesthetic:** A clean, typography-driven design utilizing `Bebas Neue` for bold headings and `DM Mono` for technical details.
- **Custom Interactions:** Features a custom JavaScript-driven cursor and fluid scrolling.
- **Scroll Reveal Animations:** Elements gracefully fade and slide into view as the user scrolls, utilizing the `IntersectionObserver` API for optimal performance.
- **Responsive Design:** Fully mobile-optimized with a custom hamburger menu and fluid grid layouts.
- **Project Case Studies:** Dedicated sub-pages (e.g., `project-zolix.html`, `project-spotnow.html`) that provide deep dives into the technical architecture and features of each application.

---

## 🛠️ Tech Stack

This portfolio is built completely from scratch without heavy frameworks to ensure maximum performance and complete design control:

- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid)
- **Interactions:** Vanilla JavaScript (ES6+)
- **Typography:** Google Fonts (`Bebas Neue`, `DM Mono`, `Outfit`)

## 🚀 Local Development

To run this project locally, simply clone the repository and open the `index.html` file in your browser, or use a live server for the best experience.

```bash
# Clone the repository
git clone https://github.com/Dax2806/Dax-Portfolio.git

# Navigate into the project directory
cd Dax-Portfolio

# (Optional) Serve with a local development server, e.g., using npx
npx serve .
```
